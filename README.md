# Data-for-optimal-sizing-of-a-standalone-charging-station-for-electric-vehicles-of-remote-communities
Data for optimal sizing of a standalone charging station for electric vehicles serving remote communities


This dataset aims to provide the parameters used in the proposed methodology, enabling transparency and reproducibility of the results. In addition, it presents a comprehensive set of figures obtained through the application of the method under different operating scenarios. The dataset includes the available photovoltaic (PV) generation for Case A and Case B, as well as the EV charging power profiles for both cases. It also reports the state of charge (SoC) of electric vehicles and charging station batteries for Case B. Finally, the optimal routing results are provided. All figures are presented in PDF format to ensure high-quality vector graphics and accurate visualization.
